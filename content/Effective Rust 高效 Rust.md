那么你已经学会了如何写一些 Rust 代码了。不过能写**一些** Rust 代码和能写**好** Rust 代码还是有区别的。

这个部分包含一些相对独立的教程，它们向你展示如何将你的 Rust 带入下一个等级。常见模式和标准库功能将被介绍。你可以选择任意顺序阅读这一部分。
